# BioCalculo

Calculadoras de aplicacao para bioensaios e ensaios de campo.

## Paginas

- `index.html`: entrada com as opcoes de calculo.
- `calda.html`: preparo de calda, PPM, dose por hectare, ajuste de ingrediente ativo e series.
- `campo.html`: calda por parcela, garrafas, bicos, CV e velocidade de passada.

Projeto estatico, pronto para GitHub Pages.
